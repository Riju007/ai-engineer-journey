# 🤖 AI Engineer Journey

> *From Python Developer to AI Engineer — one project at a time.*

## 👋 The Story Behind This Repo

I'm a Python developer with 8+ years of backend, automation, and cloud experience.

I've built production systems. I've led teams. I've shipped real products.

But AI? That was always the black box I wanted to open.

This repo is me **opening the box** — publicly, messily, and one hands-on project at a time.

No tutorials blindly followed. No copy-pasted notebooks.
Just real code, real confusion, and real learning.
